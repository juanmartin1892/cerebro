Ayúdame a escribir una spec para un nuevo script de automatización del proyecto cerebro.

Descripción del script: $ARGUMENTS

Sigue estos pasos:
1. Pregúntame las dudas necesarias para completar la spec (qué hace exactamente, qué credenciales necesita, con qué frecuencia debe ejecutarse, qué debe hacer si falla).
2. Una vez tengas suficiente información, genera el archivo `specs/[nombre-kebab-case].md` usando la plantilla en `specs/_template.md`.
3. Confirma el nombre del archivo y el cron schedule propuesto antes de crearlo.
